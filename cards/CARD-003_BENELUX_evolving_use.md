## 1 — 研究設計與方法

### 1.1 設計 [p.2]
- Retrospective, all-comers, international multicentre registry
- 7 centres in Netherlands (6) + Belgium (1)
- 倫理：Leiden Den Haag Delft MREC 豁免 (reference N22.199)；各中心 local ethics committee 核准

### 1.2 納入/排除 [p.2]
- 納入：>18 歲，接受 PCI with IVL for CAC
- 排除：無法提供 informed consent 者
- **無排除** ACS、cardiogenic shock、CTO、LM、SVG、in-stent restenosis → 與 DISRUPT-CAD 系列的重大差異

### 1.3 Endpoints 定義 [p.2-3]
| Endpoint | Definition |
|----------|------------|
| **Device success** | IVL catheter 成功到達 target lesion 並傳遞 pulses，無直接血管造影併發症 |
| **Technical success** | TIMI 3 flow + residual stenosis <30% (QCA and/or fluoroscopy) |
| **Procedural success** (primary) | Residual stenosis <30% + final TIMI 3 flow + no in-hospital MACE |
| **MACE** | Cardiac death + non-fatal target vessel MI + clinically driven TVR |
| **IVL-related complication** | 發生在 IVL 治療後立即出現的併發症，由 centralised core lab 回顧確認 |

### 1.4 影像分析 [p.2]
- Centralised core laboratory at Leiden UMC
- QCA: Medis Suite QCA (2D/3D) software V.4.0.24.4
- ICI (IVUS/OCT): QCU-CMS V.4.69
- SYNTAX Score: 由 core lab grading（三條血管皆有紀錄者）
- Calcification arc + thickness on ICI pre-IVL

### 1.5 時間趨勢分析 [p.2]
- 將收案期分為**前半期 vs. 後半期**（first/second half of inclusion period）
- 比較 ACS 比例、ICI 使用率、DCB 使用率、其他 plaque modification 使用率

### 1.6 統計 [p.3]
- Paired t-test / Wilcoxon signed-rank (paired continuous)
- Unpaired t-test / Mann-Whitney U (unpaired continuous)
- Chi-square / Fisher's exact (categorical)
- Kaplan-Meier analysis for 1-year MACE-free survival
- SPSS V.25.0; two-sided; p<0.05 significant; **未校正多重比較**

## 5 — Evolving Use Patterns (Temporal Trends)

### 5.1 趨勢分析 [p.4, Supplemental Table 1]
將收案期依時間中點分為前半期 vs. 後半期：

| Parameter | First half | Second half | p |
|-----------|-----------|-------------|---|
| ACS presentation | 33% | 48% | 0.004 |
| ICI use | 40% | 57% | 0.002 |
| DCB use | 3% | 9% | 0.021 |
| Other calcium modification | 22% | 14% | 0.034 |

### 5.2 解讀 [p.6-7]
1. **ICI 增加** → 更精確的 lesion assessment → 量身打造治療策略 → 減少額外 plaque modification 需求
2. **ACS 使用增加** → 操作者對 IVL 技術越來越熟練 → 願意在急性情境中使用
3. **DCB 增加** → "Crack and Drug" strategy 逐漸受到關注
4. **Other plaque modification 減少** → IVL 較 RA 容易使用、學習曲線短、操作時間短；IVL 可處理 deep calcium 與 eccentric lesions（RA wire bias 限制）

## 6 — 與文獻的比較 (Discussion 重點)

### 6.1 vs. DISRUPT-CAD series [p.5-6]
- DISRUPT-CAD 排除 ACS、CTO、SVG、ISR、LM、cardiogenic shock → 本研究**無排除**，涵蓋高風險族群
- 本研究 procedural success 89% vs. DISRUPT-CAD pooled (CARD-001) 92.4%（差異可能來自 all-comers 設計）
- 本研究 IVL-related complication rate 1%，與 DISRUPT-CAD 一致

### 6.2 vs. DISRUPT-CAD IV (Japanese) [p.6]
- 1-year MACE: 本研究 13% vs. DISRUPT-CAD IV 9.4%
- 差異解釋：DISRUPT-CAD IV 同樣排除 ACS、heart failure、recent MI/stroke → baseline risk 較低

### 6.3 vs. ROTAXUS trial (RA) [p.6]
- 本研究 1-year MACE 13% vs. ROTAXUS 9-month MACE 24.2%
- 提示 IVL 長期結果可能優於 RA（但為間接比較）

### 6.4 vs. REPLICA-EPIC [p.7]
- ACS + CCS 患者短期 MACE 均低，ACS 組有非顯著 MACE 上升趨勢
- 本研究 44% ACS 比例與低 MACE 率，支持 IVL 可安全用於 ACS

### 6.5 IVUS-CHIP trial [p.7]
- 提及 ICI-guided vs. angiography-guided PCI with IVL 的長期獲益正在 IVUS-CHIP trial 評估中

## 7 — Limitations [p.7]

1. **Observational design** — 無法控制混淆因素
2. **Operator discretion** — IVL 時機、predilatation/postdilatation、stent placement、ICI 使用均由術者決定
3. **ICI 非常規使用** — 限制了 standalone IVL effect 的評估，亦無法偵測 calcified nodules
4. **1-year follow-up 僅 62% 完整** — 可能低估或高估 MACE 率
5. **未校正多重比較** — 趨勢分析的 p 值可能有 type I error
6. **無法進行 lesion subtype 的詳細 outcome 分析**（scope 限制）
