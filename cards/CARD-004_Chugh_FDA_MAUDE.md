## Abstract 核心數據 [Abstract]

### 研究設計
- 資料來源：US FDA MAUDE (Manufacturer and User Facility Device Experience) database [Abstract]
- 時間範圍：2016-01-01 至 2020-12-14 [Abstract]
- 對象：所有 peripheral IVL device 的不良事件報告（含 on-label peripheral 與 off-label coronary 使用）[Abstract]

### 總體結果
- 共 **23 例**不良事件報告：20 例 peripheral artery disease + 3 例 off-label coronary [Abstract]

### Peripheral Use (n=20) [Abstract]

| 不良事件 | 發生率 |
|----------|--------|
| Device malfunction（最常見） | 13/23 (56.5%) |
| Partial balloon/catheter dislodgment（最常見 failure mode） | 12/20 (60%) |
| Balloon rupture | 3/20 (15%) |

### Off-Label Coronary Use (n=3) [Abstract]

| 不良事件 | 發生數 |
|----------|--------|
| Balloon perforation | 1/3 |
| Bradycardia | 1/3 |
| Aortocoronary dissection | 1/3 |

## 與本專案的關聯（IVL Balloon Complications）

### 核心發現

1. **Device malfunction 為最常見不良事件** (56.5%)，以 catheter/balloon dislodgment (60%) 和 balloon rupture (15%) 為主要 failure modes [Abstract]

2. **Off-label coronary IVL 使用僅 3 例報告**，但每例均有不同嚴重併發症（perforation、bradycardia、aortocoronary dissection）[Abstract]

3. **MAUDE database 為被動回報系統**，實際併發症率可能被低估（underreporting bias）[Abstract]

4. **研究時間點 (2016-2020) 早於 coronary IVL (Shockwave C2) FDA 核准**，coronary 使用均為 off-label peripheral device [Abstract]

### 限制

- MAUDE 為 passive surveillance，無法計算真正發生率（無分母數據）
- 僅 abstract 可用，無法取得各案例詳細描述
- Off-label coronary 使用樣本極少 (n=3)
- 研究期間 coronary IVL 尚未獲 FDA 核准，使用情境與現況不同

## Study Quality Notes

- FDA MAUDE database 本質為 hypothesis-generating，非 definitive evidence
- 報告偏誤（reporting bias）為此類資料庫的固有限制
- 作者結論：IVL use carries risk of complications; continued vigilance and postmarketing monitoring are warranted [Abstract]
