## Abstract 核心數據 [Abstract]

### 研究設計
- 設計：Prospective, multicenter, single-arm registry [Abstract]
- 收案期間：2018 年 8 月至 2020 年 10 月 [Abstract]
- 追蹤中位：20 months (IQR 14.5-25) [Abstract]
- 排除條件：target vessel <2.5 mm、CL 前已有 dissection [Abstract]

### 患者族群 [Abstract]

| 特徵 | 數值 |
|------|------|
| 患者數 / 病灶數 | **109 patients / 128 lesions** |
| 平均年齡 | 74 歲 |
| 糖尿病 | 58% |
| 腎功能不全 | 32% |
| 多血管疾病 (MVD) | 76% |
| STEMI 患者 | 12 例 (含 culprit calcified lesion) |

### 操作細節 [Abstract]

| 參數 | 數值 |
|------|------|
| 平均 IVL 脈衝數 | 60 pulses |
| 平均 IVL balloon 使用數 | 1 |
| 預擴張 | Semicompliant/noncompliant balloons（多數）; cutting balloon (25 例) |
| 合併 RA | 20 lesions |

### 臨床結果 [Abstract]

| 結局 | 數值 |
|------|------|
| **Procedural success** | **99%** |
| **30-day MACE-free** | **98%** |
| **Long-term MACE (median 20 mo)** | **5.6%** |

- MACE 定義：death + target-lesion revascularization (TLR) + myocardial infarction [Abstract]
- Procedural complications：few（abstract 未詳列個別項目）[Abstract]

## 與本專案的關聯（IVL Balloon Complications）

### 核心發現

1. **極高 procedural success rate (99%)** — 在高風險、真實世界族群中驗證 CL 的 feasibility [Abstract]

2. **30-day MACE-free 98%** — 短期安全性佳 [Abstract]

3. **Long-term MACE 僅 5.6%** — 在 elderly (74 yo)、高共病 (58% DM, 32% CKD, 76% MVD) 族群中的長期療效令人鼓舞 [Abstract]

4. **包含 12 例 STEMI 患者** — 少數報告 IVL 用於急性情境的 real-world data [Abstract]

5. **平均僅需 1 顆 balloon、60 pulses** — 操作相對簡單 [Abstract]

6. **合併使用 RA (20 lesions)** — 部分嚴重鈣化需 hybrid approach [Abstract]

### 限制

- 僅 abstract 可用，無法取得各項 procedural complications 細節（perforation、dissection、balloon rupture 等）
- Single-arm 設計，無對照組
- 109 patients 樣本量有限，無法偵測罕見併發症
- 收案期早期 (2018-2020)，IVL 經驗可能處於 learning curve
- Abstract 未報告 balloon-specific adverse events

## Study Quality Notes

- 作者聲稱為首個 real-world multicenter registry 提供 CL 長期追蹤數據 [Abstract]
- Prospective 收案增加可信度
- High-risk population（elderly, DM, CKD, MVD）增加外部效度
- 包含 STEMI 為獨特價值
- 作者結論：confirms safety and long-term efficacy of PCI with CL in unselected, high-risk population [Abstract]
